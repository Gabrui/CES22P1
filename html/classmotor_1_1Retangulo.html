Construtores & Destrutores

def motor.Retangulo.__init__ (   self,
  ponto1 = None,
  ponto2 = None,
  largura = 0,
  altura = 0 
)

init Inicializa o retângulo com dois pontos ou com um ponto e uma largura \ e uma altura.

O retângulo é horizontal e não pode ser rotacionado.\ O retângulo utiliza as coordenadas cartesianas com x positivo \ para direita e y positivo para baixo.

Parâmetros
{Ponto}ponto1 Representa um vértice do retângulo. Ele é usado \ como ponto superior esquerdo se não houver
{Ponto}ponto2 Representa o outro vértice diametralmente oposto
{float}largura É ignorado se houver ponto2
{float}altura É ignorado se houver ponto2
